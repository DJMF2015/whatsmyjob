Whatsmyjob? : Android app

An occupational evaluation app for school leavers designed to help suggest possible career suggestions based upon the questionnaire

Uses an algorithm to produce a set of 'top ten jobs' and 'top five' highest ranking work environments best suited to the user based uponn their answers provided in the app

Designed as dissertation project for MSc IT, Stirling University 2016
